13: 
14: static string ReverseString(string input)
15: {
16:     // 最好不用直接用 string 进行拼接 (string vs StringBuilder)
17:     string result = "";
18:     foreach (char c in input)
19:     {
20:         result = c + result;
21:     }
22: 
23:     return result;
24: }

改进:

  1. 优化代码，让其更加高效合理 (stringbuilder)

2.1.5. 在控制台上，输出倒三角图案

*****
****
***
**
*

实现思路:

  • 先写一个简单的循环，输出 N 行，每行一个 *
  • 尝试修改上述代码，输出 N 行，每行输出 M 个星
  • 再修改上述代码，让次循环中输出的星的个数跟当前行数关联
参考答案 
1: for (int i = 0; i < 9; i++)
2: {
3:     for (int j = 0; j < 9 - i; j++)
4:     {
5:         Console.Write("*");
6:     }
7:     Console.WriteLine();
8: }

3. 知识点

3.1. 字符串的拼接

简而言之，字符串有如下 4 种常用的拼接方式。代码示例如下: 

 1: string a = "hello";
 2: string b = "world";
 3: string c = "!";
 4: 
 5: // 拼接起来
 6: string r1 = a + b + c; // 语法简单自然
 7: string r2 = new StringBuilder().Append(a).Append(b).Append(c).ToString(); // 效率更高 (✓)
 8: string r3 = string.Format("{0}{1}{2}", a, b, c); // 结合了前面两种方式的优点
 9: string r4 = $"{a}{b}{c}";                        // 是在 string.Format 基础上进一步优化语法 (✓)
10: 
11: Console.WriteLine(r1);
12: Console.WriteLine(r2);
13: Console.WriteLine(r3);
14: Console.WriteLine(r4);

String vs StringBuilder, 效率差别。请执行下面两段代码，进行观察。 

1: string s = null;
2: 
3: for (int i = 0; i < 1000000; i++)
4: {
5:     s = s + "hello.";
6: }
7: 
8: Console.WriteLine(s);

1: StringBuilder s = new StringBuilder();
2: 
3: for (int i = 0; i < 1000000; i++)
4: {
5:     s = s.Append("hello.");
6: }
7: 
8: Console.WriteLine(s);
