2.2. Live Server 用来实时刷新 html 页面

首先，在 插件 界面，输入 live，可以看到 Live Server 的插件。

点击下载、安装。

如果想让当前编辑的 html 页面能够实时刷新。需要:

  1. 确保在 VSCode 中，已经将当前文件夹打开。如果没有，点击打开

  2. 在 html 的编辑界面，右键点击打开 *Live Server*，会自动弹出页面。
