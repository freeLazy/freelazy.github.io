1. 快捷键

全局：

  • Win-r 打开【运行】窗口
  • Win-e 打开【文件管理器】

记事本中：

  • Ctrl-s 保存文本内容
  • Ctrl-z 撤销最近一次修改
  • Ctrl-c/v 复制、粘贴

2. 命令提示符

打开方式:

  1. 点击开始菜单 -> 输入 cmd 回车 -> 通过 t: 切换盘符，通过 cd 切换文件夹
  2. 快捷键 Win-r 后输入 cmd 回车 -> 通过 t: 切换盘符，通过 cd 切换文件夹
  3. 在文件管理器的地址栏，输入 cmd 直接打开命令提示符，并且当前工作目录就是文件夹位置

基本规则:

  • 敲入命令，然后敲回车执行
  • 命令类似 cp a b 这样的格式，使用空格分割，第一部分 (cp) 是命令程序，其他部分是参数。参数可以有 0 个或多个
