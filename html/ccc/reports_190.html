3. 寒假作业 (2024-01-15)

做一个求取 BMI 的控制台应用:

  1. 首先，提示用户输入身高和体重 (Console.ReadLine)
  2. 其次，根据用户输入的身高和体重，计算 BMI，并判定身体状态 (if..else)
  3. 最后，输出给用户 (Console.WriteLine)
  4. 进入第一步，循环 (for/while)

参考网站: https://cn.onlinebmicalculator.com/

其他:

  • 如果有兴趣的同学，可以自行学习一下 Winform 应用的创建，尝试使用窗口的方式将这个功能实现一下

4.1. 实现一个求取 BMI 值的控制台应用

基本实现: 

static void Main(string[] args)
{
    Console.WriteLine("欢迎使用 BMI 计算器！\n");
    // 获取身高:
    // 1）通过 Console.ReadLine 方法读取用户输入，结果是一个字符串
    // 2) 通过 double.Parse 方法可以将字符串转换为 double 类型
    Console.Write("请输入您的身高（单位：米）: ");
    double shengao = double.Parse(Console.ReadLine());
    // 获取体重:
    Console.Write("请输入您的体重（单位：千克）: ");
    double tizhong = double.Parse(Console.ReadLine());

    // 通过 shengao/tizhong 来求取 BMI 值
    // 计算公式: 体重/身高^2
    double bmi = tizhong / (shengao * shengao);

    // 输出 BMI
    // 通过变量，存储中间数据
    // 通过字符串拼接+Console.WriteLine 反馈给用户
    Console.WriteLine($"\n您的身高是 {shengao}，您的体重是 {tizhong}，您的 Bmi 为 {bmi}");
}
